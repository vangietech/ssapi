## Windows 客户端使用方法： <br>
![](./images/winSetup.png) <br>
如上图填写好账号信息，点击确定。 <br>
![](./images/winConnect.png) <br>
这时客户端会隐藏在右下角，点击启动系统代理。 <br> 
注意：系统代理模式一定要选择PAC模式，否则我们正常访问国内的网站也会经过代理，这样会减慢国内网页的访问速度。 <br>
